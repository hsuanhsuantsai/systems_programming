# named_pipe
Implement named pipe

## Usage
* How to compile:  
  - Use `make` with makefile
* How to run:  
  - ./my_npipe_writer
  - ./my_npipe_reader
  
## Running example
1. ./my_npipe_writer
2. (User enters desired messages) Enter Input: `Hello world`
3. Enter `exit` to finish writing
4. ./my_npipe_reader
5. Should see `Hello world`
